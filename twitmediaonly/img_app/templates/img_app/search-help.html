Search Help

Rushview has two independent search fields that query the Twitter API.

Search by @Username

"Search by @Username" to search by a Twitter username and return all that user's media tweets.

Example search:

@ladygaga

General Search

Use "General Search" as you would for a typical search on Twitter. For a more detail on building a twitter query, see the Twitter Guide on building a search query. The Twitter API currently limits search results to tweets posted within the last week.

This search supports basic boolean operators (AND, OR, NOT, Grouping, Negations). Some example searches:

#dog

apple OR (iphone ipad)

(happy OR happiness) lang:en -birthday -is:retweet

(happy OR happiness) place_country:GB -birthday -is:retweet
